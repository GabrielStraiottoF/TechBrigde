# TechBridge - Plataforma de Oportunidades para Alunos Técnicos

## 📋 Descrição

TechBridge é um website moderno e responsivo desenvolvido para servir como uma ponte conectando alunos que saem do ensino médio técnico com oportunidades de:

- 🎓 **Faculdades e Universidades** - Explorando caminhos acadêmicos após o técnico
- 💼 **Oportunidades de Emprego** - Encontrando a primeira posição profissional
- 🚀 **Empreendedorismo** - Guias para abrir o próprio negócio

## 🎯 Funcionalidades Principais

### 1. **Navegação Intuitiva**
- Menu responsivo que se adapta em dispositivos móveis
- Navegação suave entre seções
- Header sticky para acesso rápido

### 2. **Hero Section**
- Apresentação atrativa com gradiente moderno
- Botões de chamada à ação
- Animações de entrada

### 3. **Seção de Faculdades**
- Listagem de instituições recomendadas (USP, UNICAMP, FATEC)
- Informações sobre cursos disponíveis
- Botões para explorar mais

### 4. **Oportunidades de Emprego**
- 3 vagas de exemplo com detalhes realistas
- Faixa salarial
- Requisitos e competências
- Botão de candidatura

### 5. **Guia de Empreendedorismo**
- 6 passos práticos para iniciar um negócio
- Design numerado e didático
- Links para mais informações em cada etapa

### 6. **Newsletter**
- Inscrição para receber atualizações
- Validação de email
- Design destacado

### 7. **Formulário de Contato**
- Campos para nome, email e mensagem
- Informações de contato direto
- Validação de formulário

## 🛠️ Tecnologias Utilizadas

## 🚀 Como Usar

1. **Baixar os arquivos**: Certifique-se de ter os 3 arquivos na mesma pasta:
   - `index.html`
   - `style.css`
   - `script.js`

2. **Abrir no navegador**: 
   - Clique duplo no `index.html` ou
   - Arraste o arquivo para o navegador ou
   - Use um servidor local como Live Server

3. **Navegar**:
   - Use o menu para navegar entre seções
   - Clique nos botões de ação
   - Preencha os formulários (demonstração)
